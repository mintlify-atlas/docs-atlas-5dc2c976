# Quick start

This guide will help you integrate the Animated Profile Slider into your Flutter app and get a working carousel in minutes.

## Basic setup

The simplest way to use the carousel is to add it to your app's main widget tree.

<Steps>
  <Step title="Import the widget">
    Import the animated slider in your main app file:

    ```dart
    import 'package:flutter/material.dart';
    import 'package:flutter/services.dart';
    ```
  </Step>

  <Step title="Configure orientation (optional)">
    For optimal display, the carousel works best in landscape mode. You can force landscape orientation in your `main()` function:

    ```dart
    void main() {
      WidgetsFlutterBinding.ensureInitialized();
      // Force landscape orientation for better carousel display
      SystemChrome.setPreferredOrientations([
        DeviceOrientation.landscapeLeft,
        DeviceOrientation.landscapeRight,
      ]).then((_) {
        runApp(const MyApp());
      });
    }
    ```

    This ensures users see the full carousel width without rotation.
  </Step>

  <Step title="Add the carousel to your widget tree">
    Use the `SliceAnimatedList` widget in your app:

    ```dart
    class MyApp extends StatelessWidget {
      const MyApp({super.key});

      @override
      Widget build(BuildContext context) {
        return MaterialApp(
          title: 'Animated Profile Slider',
          theme: ThemeData(
            primarySwatch: Colors.blue,
            brightness: Brightness.dark,
          ),
          home: const Scaffold(
            body: SliceAnimatedList(),
          ),
        );
      }
    }
    ```
  </Step>
</Steps>

## Understanding the code

Here's how the carousel widget works internally:

### Initialization

The widget initializes with 5 items by default and sets up two key components:

```dart
@override
void initState() {
  super.initState();
  // Initialize the carousel with 5 items
  _populateInitialItems();
  
  // Setup button animation controller for pulsing effect
  _initializeButtonAnimation();
}
```

The `_populateInitialItems()` method creates the initial item list, while `_initializeButtonAnimation()` sets up the pulsing animation for navigation buttons.

### Item selection and animation

When you select an item (either by tapping or using navigation buttons), the widget updates its state:

```dart
void _selectItem(int index) {
  setState(() {
    _selectedIndex = index;
  });
}
```

This triggers a rebuild where each item's position and scale are recalculated based on distance from the selected index.

### Scale calculation

The widget uses a distance-based algorithm to determine each item's scale:

```dart
double _calculateItemScale(int index) {
  final int distanceFromCenter = (index - _selectedIndex).abs();
  return (_maxScale - (distanceFromCenter * _scaleFactor)).clamp(
    _minScale,
    _maxScale,
  );
}
```

- Selected item (distance = 0): 1.0x scale (full size)
- One item away (distance = 1): 0.8x scale
- Two items away (distance = 2): 0.6x scale
- Three or more items away: 0.3x scale (minimum)

### Animation timing

All position and scale transitions use consistent timing:

```dart
AnimatedPositioned(
  duration: Duration(milliseconds: 300),
  curve: Curves.easeInOut,
  // ... positioning logic
  child: AnimatedScale(
    scale: scale,
    duration: Duration(milliseconds: 300),
    curve: Curves.easeInOut,
    // ... item widget
  ),
)
```

The 300ms duration with `easeInOut` curve provides smooth, natural-feeling transitions.

## Complete example

Here's a complete working example that includes everything:

```dart
import 'package:flutter/material.dart';
import 'package:flutter/services.dart';

void main() {
  WidgetsFlutterBinding.ensureInitialized();
  SystemChrome.setPreferredOrientations([
    DeviceOrientation.landscapeLeft,
    DeviceOrientation.landscapeRight,
  ]).then((_) {
    runApp(const MyApp());
  });
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Animated Profile Slider',
      theme: ThemeData(
        primarySwatch: Colors.blue,
        brightness: Brightness.dark,
      ),
      home: const Scaffold(
        body: SliceAnimatedList(),
      ),
    );
  }
}
```

<Info>
The carousel automatically shows navigation buttons only when needed. Left arrow appears when you can go left, right arrow when you can go right.
</Info>

## Navigation controls

The carousel provides two ways to navigate:

1. **Tap any item** - Directly select it by tapping the card
2. **Use arrow buttons** - Click the pulsing left/right arrows to move through items sequentially

Navigation buttons automatically hide at the boundaries:

```dart
// Left navigation button - only show if not at first item
if (_selectedIndex > 0)
  Positioned(
    left: 20,
    child: _buildNavigationButton(
      icon: Icons.arrow_back_ios,
      onPressed: _selectPreviousItem,
      isLeft: true,
    ),
  ),
```

## Customization constants

The widget defines several constants you can adjust to customize behavior:

```dart
static const double _itemWidth = 150.0;        // Width of each card
static const double _itemSpacing = 180.0;      // Space between item centers
static const double _maxScale = 1.0;           // Scale of selected item
static const double _minScale = 0.3;           // Minimum scale for distant items
static const double _scaleFactor = 0.2;        // Scale reduction per item distance
static const Duration _animationDuration = Duration(milliseconds: 300);
static const Duration _buttonPulseDuration = Duration(milliseconds: 1500);
```

<Tip>
To customize animations, modify these constants in the `_SliceAnimatedListState` class. For example, increase `_animationDuration` to 500ms for slower transitions.
</Tip>
