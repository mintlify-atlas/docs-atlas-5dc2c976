## Constants

The `_SliceAnimatedListState` class defines several private constants that control the carousel's behavior and appearance.

### Layout constants

<ResponseField name="_itemWidth" type="double" default="150.0">
  Width of each carousel item card in logical pixels. This defines the horizontal size of individual cards.
  
  ```dart
  static const double _itemWidth = 150.0;
  ```
</ResponseField>

<ResponseField name="_itemSpacing" type="double" default="180.0">
  Horizontal spacing between consecutive items in logical pixels. Controls how far apart items appear when laid out in the carousel.
  
  ```dart
  static const double _itemSpacing = 180.0;
  ```
</ResponseField>

### Scale constants

<ResponseField name="_maxScale" type="double" default="1.0">
  Maximum scale factor for the selected item. The centered/selected item is displayed at full size (100%).
  
  ```dart
  static const double _maxScale = 1.0;
  ```
</ResponseField>

<ResponseField name="_minScale" type="double" default="0.3">
  Minimum scale factor for items far from selection. Items scale down to 30% of their original size when distant from the selected item.
  
  ```dart
  static const double _minScale = 0.3;
  ```
</ResponseField>

<ResponseField name="_scaleFactor" type="double" default="0.2">
  Scale reduction per step from the selected item. Each position away from the center reduces scale by 0.2 (20%).
  
  ```dart
  static const double _scaleFactor = 0.2;
  ```
  
  <Expandable title="Scale calculation example">
  
  For a carousel with item at index 2 selected:
  - Index 2 (selected): scale = 1.0
  - Index 1 or 3 (1 step away): scale = 0.8
  - Index 0 or 4 (2 steps away): scale = 0.6
  - Index beyond (3+ steps): scale = 0.4 or clamped to 0.3 minimum
  
  </Expandable>
</ResponseField>

### Animation duration constants

<ResponseField name="_animationDuration" type="Duration" default="300 milliseconds">
  Duration for item position and scale transitions. Controls how long it takes for items to move and resize when selection changes.
  
  ```dart
  static const Duration _animationDuration = Duration(milliseconds: 300);
  ```
</ResponseField>

<ResponseField name="_buttonPulseDuration" type="Duration" default="1500 milliseconds">
  Duration for one complete pulse cycle of the navigation buttons. The buttons smoothly scale up and down over 1.5 seconds.
  
  ```dart
  static const Duration _buttonPulseDuration = Duration(milliseconds: 1500);
  ```
</ResponseField>

## State variables

These mutable properties track the current state of the carousel and manage animations.

### Data and selection state

<ResponseField name="_items" type="List<int>" default="[]">
  List storing the carousel items. Each integer represents a unique item identifier used for display and color selection.
  
  ```dart
  final List<int> _items = [];
  ```
  
  <Note>
  Items are populated during initialization with values from `_counter`. The list grows as items are added to the carousel.
  </Note>
</ResponseField>

<ResponseField name="_counter" type="int" default="0">
  Counter for generating unique item identifiers. Increments each time a new item is added to the carousel.
  
  ```dart
  int _counter = 0;
  ```
</ResponseField>

<ResponseField name="_selectedIndex" type="int" default="2">
  Index of the currently selected item in the carousel. Defaults to 2 (middle of the initial 5 items).
  
  ```dart
  int _selectedIndex = 2; // Default selected item (middle of initial 5 items)
  ```
  
  <Tip>
  This property drives all positioning and scaling calculations. Changing this value triggers animations to reposition all items relative to the new center.
  </Tip>
</ResponseField>

### Animation state

<ResponseField name="_buttonAnimationController" type="AnimationController" required>
  Controller managing the pulse animation for navigation buttons. Initialized in `initState()` with `_buttonPulseDuration`.
  
  ```dart
  late AnimationController _buttonAnimationController;
  ```
  
  <Expandable title="Initialization details">
  
  ```dart
  _buttonAnimationController = AnimationController(
    vsync: this,
    duration: _buttonPulseDuration,
  )..repeat(reverse: true);
  ```
  
  The controller uses `repeat(reverse: true)` to create a continuous back-and-forth animation.
  
  </Expandable>
</ResponseField>

<ResponseField name="_buttonAnimation" type="Animation<double>" required>
  Animation that drives the scale transformation for navigation buttons. Maps controller value to scale range 1.0-1.3.
  
  ```dart
  late Animation<double> _buttonAnimation;
  ```
  
  <Expandable title="Animation configuration">
  
  ```dart
  _buttonAnimation = Tween<double>(begin: 1.0, end: 1.3).animate(
    CurvedAnimation(
      parent: _buttonAnimationController,
      curve: Curves.easeInOut,
    ),
  );
  ```
  
  The animation smoothly scales buttons between 100% and 130% of their size using an easeInOut curve.
  
  </Expandable>
</ResponseField>

## Property relationships

```dart
// Scale calculation uses multiple constants:
double _calculateItemScale(int index) {
  final int distanceFromCenter = (index - _selectedIndex).abs();
  return (_maxScale - (distanceFromCenter * _scaleFactor)).clamp(
    _minScale,
    _maxScale,
  );
}

// Position calculation uses spacing constant:
final double offsetX = (index - _selectedIndex) * _itemSpacing;
```

<Note>
All constants use `lowerCamelCase` with underscore prefix following Flutter conventions for private class constants.
</Note>
