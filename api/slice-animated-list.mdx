## Overview

`SliceAnimatedList` is the core widget of the Animated Profile Slider package. It creates a horizontal carousel with smooth animations where items scale based on their distance from the selected item.

## Key features

- **Scale-based selection** - Selected item scales to full size (1.0) while others scale down
- **Smooth animations** - 300ms transitions with easeInOut curves
- **Interactive navigation** - Tap to select items or use arrow buttons
- **Pulsing navigation buttons** - 1.5s pulse animation to draw attention
- **Dynamic positioning** - Items automatically position relative to center
