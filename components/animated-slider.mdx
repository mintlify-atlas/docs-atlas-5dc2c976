## Overview

The `SliceAnimatedList` is the main carousel component that displays items in a horizontal row with the selected item centered and scaled up. It provides smooth animations when navigating between items and automatically manages item positioning and scaling.

<Info>
The widget uses `StatefulWidget` with `TickerProviderStateMixin` to support animation controllers.
</Info>

## Architecture

The component is built as a stateful widget that manages its own animation state:

```dart
class SliceAnimatedList extends StatefulWidget {
  const SliceAnimatedList({super.key});

  @override
  _SliceAnimatedListState createState() => _SliceAnimatedListState();
}

class _SliceAnimatedListState extends State<SliceAnimatedList>
    with TickerProviderStateMixin {
  // State management and animation logic
}
```

### Key components

The architecture relies on `TickerProviderStateMixin` to provide the `vsync` parameter for animation controllers, enabling efficient animation performance by syncing with the screen refresh rate.

## State management

The widget manages three key pieces of state:

### Item list

```dart
final List<int> _items = [];
int _counter = 0;
```

The `_items` list stores all carousel items (represented as integers), while `_counter` tracks the next item ID to add.

### Selected index

```dart
int _selectedIndex = 2; // Default selected item (middle of initial 5 items)
```

The `_selectedIndex` determines which item is centered and scaled to full size. It defaults to `2` to center the middle item when the carousel initializes with 5 items.

### Button animation controller

```dart
late AnimationController _buttonAnimationController;
late Animation<double> _buttonAnimation;
```

These manage the pulsing animation for navigation buttons.

## Initialization

The widget initializes in two steps:

### Populating initial items

```dart
void _populateInitialItems() {
  for (int i = 0; i < 5; i++) {
    _items.add(_counter++);
  }
}
```

Creates 5 initial items numbered 0-4.

### Setting up button animation

```dart
void _initializeButtonAnimation() {
  _buttonAnimationController = AnimationController(
    vsync: this,
    duration: _buttonPulseDuration,
  )..repeat(reverse: true);

  _buttonAnimation = Tween<double>(begin: 1.0, end: 1.3).animate(
    CurvedAnimation(
      parent: _buttonAnimationController,
      curve: Curves.easeInOut,
    ),
  );
}
```

Creates a repeating scale animation from 1.0 to 1.3 with smooth easing.

## How it works

The carousel follows a centered selection model:

1. **Item positioning**: All items are positioned relative to the screen center, with the selected item at center
2. **Scale calculation**: Each item's scale is calculated based on its distance from the selected index
3. **Animation**: When selection changes, items animate to their new positions and scales
4. **Navigation**: Left/right buttons appear conditionally and allow moving through the list

### Selection logic

```dart
void _selectItem(int index) {
  setState(() {
    _selectedIndex = index;
  });
}
```

<Tip>
Calling `setState()` triggers a rebuild, which recalculates all item positions and scales based on the new `_selectedIndex`.
</Tip>

## Widget structure

The main build method creates a stack-based layout:

```dart
@override
Widget build(BuildContext context) {
  return Column(
    children: <Widget>[
      SizedBox(
        height: 150.0,
        width: MediaQuery.of(context).size.width,
        child: Stack(
          alignment: Alignment.center,
          children: [
            // Generate all carousel items
            ...List.generate(
              _items.length,
              (index) => _buildCarouselItem(context, index),
            ),

            // Left navigation button - only show if not at first item
            if (_selectedIndex > 0)
              Positioned(
                left: 20,
                child: _buildNavigationButton(
                  icon: Icons.arrow_back_ios,
                  onPressed: _selectPreviousItem,
                  isLeft: true,
                ),
              ),

            // Right navigation button - only show if not at last item
            if (_selectedIndex < _items.length - 1)
              Positioned(
                right: 20,
                child: _buildNavigationButton(
                  icon: Icons.arrow_forward_ios,
                  onPressed: _selectNextItem,
                  isLeft: false,
                ),
              ),
          ],
        ),
      ),
    ],
  );
}
```

### Layout breakdown

<Accordion title="Column wrapper">
  The root `Column` widget allows for potential expansion with additional UI elements above or below the carousel.
</Accordion>

<Accordion title="Fixed height container">
  A `SizedBox` with 150.0 height and full width constrains the carousel area.
</Accordion>

<Accordion title="Stack with centered alignment">
  All carousel items and navigation buttons are positioned within a `Stack` that uses center alignment as the reference point.
</Accordion>

<Accordion title="Dynamic item generation">
  The spread operator (`...`) expands a list of carousel items generated by `List.generate`.
</Accordion>

<Accordion title="Conditional navigation buttons">
  Left and right buttons only appear when navigation in that direction is possible.
</Accordion>

## Cleanup

The widget properly disposes of animation controllers:

```dart
@override
void dispose() {
  _buttonAnimationController.dispose();
  super.dispose();
}
```

<Warning>
Always dispose of animation controllers to prevent memory leaks.
</Warning>
